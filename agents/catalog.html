Agent Catalog

Official agents for common server-side patterns. Free agents ship with every Weblisk server implementation.

Infrastructure Agents Free

System-level services available to all domains. Ship with every Weblisk hub.

AgentKindDescriptionTriggers
sync Infrastructure Background data synchronisation between client-side IDB store and server database. Supports conflict resolution strategies. Event + Schedule
cron Infrastructure Scheduled task execution with cron-style expressions. Supports one-time and recurring tasks with retry logic. Schedule
webhook Infrastructure Inbound and outbound webhook processing with signature verification, retries, and delivery tracking. Event (HTTP)
email-send Infrastructure Transactional email with template rendering, queuing, and delivery status tracking. Supports SMTP and API-based providers. Event

Domain Work Agents Free

Specialized agents dispatched by domain controllers. Each agent serves a specific business function.

SEO Domain

AgentKindDescriptionTriggers
seo-analyzer Work Audits pages for title tags, meta descriptions, heading hierarchy, keyword density, and structured data. Domain dispatch
a11y-checker Work Validates accessibility compliance — ARIA roles, color contrast, alt text, focus management, and WCAG guidelines. Domain dispatch

Content Domain

AgentKindDescriptionTriggers
content-analyzer Work Measures readability (Flesch-Kincaid), heading structure, paragraph balance, and content quality signals. Domain dispatch
meta-checker Work Validates HTML metadata completeness — Open Graph, Twitter cards, structured data (JSON-LD), and canonical URLs. Domain dispatch

Health Domain

AgentKindDescriptionTriggers
uptime-checker Work Checks endpoint availability, response time, DNS resolution, and TLS certificate status with grading. Domain dispatch
perf-auditor Work Measures page load performance, Core Web Vitals (LCP, FID, CLS), resource budgets, and optimization opportunities. Domain dispatch
