Accessible Tabs

ARIA-compliant tablist with keyboard navigation (Arrow keys, Home, End).

Overview: Tabs organize content into sections that users switch between. Each panel is only visible when its tab is active. Fully keyboard-navigable with proper ARIA roles.
API: tabs(container, { defaultIndex, onChange }) — Returns { activate, active }. Active tab index is a signal you can observe with effect().
Usage: import { tabs } from 'weblisk/pro/tabs.js'; then call tabs('[data-tabs]'). Tabs auto-bind to [role="tab"] elements inside the container.
