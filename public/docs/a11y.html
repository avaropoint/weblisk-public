Accessibility

ARIA announcements, focus traps, roving tabindex, reduced motion detection, and locale-aware formatting. Build inclusive UIs with reactive signals.

ARIA Live Regions

Announce messages to screen readers via a managed ARIA live region. Polite announcements wait for the user to finish reading; assertive ones interrupt immediately.

ARIA Announcements

Screen reader announcements via ARIA live region.

No announcements yet
import { announce, aria, role } from 'weblisk/a11y/aria.js';

// Polite announcement (waits for user)
announce('3 new items loaded');

// Assertive announcement (interrupts)
announce('Error: Connection lost', 'assertive');

// Declarative ARIA attributes
aria('#dialog', { modal: true, labelledby: 'dialog-title' });

// Set a role with required attributes
role('#tab-panel', 'tabpanel', { labelledby: 'tab-1' });

ARIA API

ExportTypeDescription
announce(msg, priority?)functionAnnounce to screen readers — 'polite' (default) or 'assertive'
aria(sel, attrs)functionSet ARIA attributes declaratively on an element
role(sel, role, attrs?)functionSet role and required ARIA attributes

Focus Management

Focus traps for modals/dialogs, roving tabindex for toolbars/menus, and skip-to-content links for keyboard navigation.

Focus Management

Focus trap + roving tabindex. Click "Open Trap" then Tab to cycle.

import { focusTrap, rovingIndex, skipLink } from 'weblisk/a11y/focus.js';

// Trap focus within a container
const trap = focusTrap('#modal');
trap.activate();   // Tab cycles within #modal
trap.deactivate(); // Release focus

// Arrow-key navigation for toolbars
rovingIndex('#toolbar', 'button', {
  horizontal: true, wrap: true
});

// Skip-to-content link
skipLink('main-content');

Focus API

ExportTypeDescription
focusTrap(el)functionCreate a focus trap — returns { activate, deactivate }
rovingIndex(container, sel, opts?)functionArrow-key navigation within a group
skipLink(targetId)functionInject a "Skip to main content" link

Reduced Motion

Respect the user's motion preference with a reactive signal. Conditionally disable animations and transitions.

Reduced Motion

Detects prefers-reduced-motion. Toggle in OS settings to see it change live.

Checking...
import { motionReduced, motion, safeTransition, ifMotion } from 'weblisk/a11y/motion.js';

// Reactive signal
effect(() => console.log('Reduced:', motionReduced()));

// Choose value based on preference
const dur = motion('0.3s', '0s');

// Only set transition if motion is allowed
safeTransition('#panel', 'transform 0.3s ease');

// Run code only if motion is allowed
ifMotion(() => el.animate([{opacity:0},{opacity:1}], 300));

Motion API

ExportTypeDescription
motionReduced()signalReactive boolean — true when user prefers reduced motion
motion(full, reduced)functionReturn full or reduced based on preference
safeTransition(sel, transition)functionSet CSS transition only when motion is allowed
ifMotion(fn)functionRun a function only when motion is allowed
injectReducedMotionCSS()functionInject global CSS that disables all animations when reduced

Locale & Internationalization

Signal-driven locale detection and Intl API formatting wrappers — no translation dictionaries needed.

js 
import { locale, setLocale, textDirection, formatter } from 'weblisk/a11y/locale.js';

console.log(locale());         // 'en-US'
setLocale('fr');               // Switch — formatters update reactively
textDirection();             // 'ltr' or 'rtl'

const fmt = formatter();
fmt.number(1234.5);            // '1,234.5' or '1 234,5'
fmt.date(new Date());           // locale-formatted date
fmt.currency(99.99, 'USD');     // '$99.99' or '99,99 $US'
fmt.relative(-2, 'day');       // '2 days ago'
fmt.list(['A', 'B', 'C']);      // 'A, B, and C'
